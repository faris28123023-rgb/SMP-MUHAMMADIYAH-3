Program Keahlian

Pilihan jurusan yang relevan dengan industri teknologi masa kini

Rekayasa Perangkat Lunak (RPL)

Jurusan ini mempelajari cara membuat, memelihara, dan mengelola perangkat lunak (software) menggunakan berbagai bahasa pemrograman yang paling banyak digunakan di industri.

Materi Pokok:

  • Pemrograman Dasar (C++, Python)
  • Pengembangan Web (HTML, CSS, JS, PHP, Framework)
  • Pemrograman Mobile (Android/iOS)
  • Manajemen Basis Data (MySQL, MongoDB)

Prospek Karier: Web Developer, Software Engineer, Programmer, Analis Sistem.

Teknik Komputer dan Jaringan (TKJ)

Jurusan TKJ berfokus pada perangkat keras komputer dan infrastruktur jaringan, dari merakit komputer hingga membangun server untuk skala perusahaan besar.

Materi Pokok:

  • Instalasi dan Perakitan Komputer
  • Administrasi Server (Linux & Windows Server)
  • Desain dan Manajemen Jaringan (Cisco, Mikrotik)
  • Keamanan Jaringan Komputer

Prospek Karier: Network Engineer, IT Support, Administrator Jaringan, System Admin.

Multimedia (Desain Komunikasi Visual)

Jurusan ini mengajarkan kreativitas visual yang dipadukan dengan teknologi. Siswa akan belajar mendesain, memproduksi video, hingga membuat animasi interaktif.

Materi Pokok:

  • Desain Grafis (Photoshop, Illustrator)
  • Fotografi dan Videografi Dasar
  • Animasi 2D dan 3D
  • Desain Antarmuka (UI/UX Design)

Prospek Karier: Graphic Designer, Video Editor, Animator, Content Creator.
